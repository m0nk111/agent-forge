## Advanced Example

Here's an advanced example of how to use the library:

1. Import the necessary module:
    ```python
    import my_library
    ```

2. Initialize the library with your API key:
    ```python
    api = my_library.API(api_key='your_api_key')
    ```

3. Use a specific feature from the library:
    ```python
    result = api.fetch_data(query='example query')
    print(result)
    ```
