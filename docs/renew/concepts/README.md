# Core Concepts

This section provides an overview of the core concepts in the Agent-Forge platform.

## Key Concepts

- **Agent**: An autonomous entity that can perform tasks.
- **Task**: A unit of work that can be assigned to an agent.
- **Service**: A long-running process that provides a specific functionality.
- **Pipeline**: A sequence of tasks that are executed in a specific order.
- **Coordinator**: An agent responsible for orchestrating the work of other agents.
